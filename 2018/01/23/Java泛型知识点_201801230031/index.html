发表于 2018-01-23

泛型的好处

 泛型的好处就是让编译器保留参数的类型信息，在执行类型检查、类型转换（casting）操作时，让编译器保证了这些类型转换（casting）的绝对无误（保证类型安全）。

泛型的类型擦除

 Java中的泛型只存在于编译期，在将源文件编译完成的Java字节码中是不包含泛型中的类型信息的。使用泛型的时候加上类型参数，会被编译器在编译的时候擦除掉。这个过程称为“类型擦除（type erasure）”。如：
