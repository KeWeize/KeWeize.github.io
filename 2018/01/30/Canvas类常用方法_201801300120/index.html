Canvas类常用方法

发表于 2018-01-30

Canvas方法

填充画布，这类颜色填充方法一般用于在绘制之前设置底色，或者在绘制之后为界面设置半透明蒙版

drawARGB(int a, int r, int g, int b)
用指定的ARGB颜色填充整个画布位图

drawColor(int color)
用指定的颜色填充整个画布位图

drawRGB(int r, int g, int b)
用指定的RGB颜色填充整个画布位图

drawPaint(Paint paint)
用指定的油漆填充整个画布的位图(仅限于当前剪辑)

画圆

drawCircle(float cx, float cy, float radius, Paint paint)
使用指定的Paint对象绘制指定的圆

画矩形

drawRect(Rect r, Paint paint)
使用指定的Paint对象绘制指定的矩形区域。

drawRect(RectF rect, Paint paint)
使用指定的Paint对象绘制指定的矩形区域。

画圆角矩形

drawRoundRect(RectF rect, float rx, float ry, Paint paint)
使用指定的Paint对象制指定的圆形矩形。

drawRoundRect(float left, float top, float right, float bottom, float rx, float ry, Paint paint)
使用指定的Paint对象制指定的圆形矩形。

画点

drawPoint(float x, float y, Paint paint)
用于绘制单个点的drawPoints()的帮助器

drawPoints(float[] pts, Paint paint)
用于drawPoints()的Helper函数，它假定您希望绘制整个数组（pts : [x0 y0 x1 y1 x2 y2 …]）

drawPoints(float[] pts, int offset, int count, Paint paint)
画出一系列的点

画椭圆

drawOval(float left, float top, float right, float bottom, Paint paint)
使用指定的Paint对象绘制指定的椭圆形

drawOval(RectF oval, Paint paint)
使用指定的Paint对象绘制指定的椭圆形

画线

drawLine(float startX, float startY, float stopX, float stopY, Paint paint)
使用指定的Paint对象在指定的起始点~终点绘制一条线段

drawLines(float[] pts, int offset, int count, Paint paint)
绘制一系列线 （pts说明:Array of points to draw [x0 y0 x1 y1 x2 y2 …]；
offset为跳过的点，count为要绘制的点数量）

drawLines(float[] pts, Paint paint)
绘制一系列线

画弧

drawArc(float left, float top, float right, float bottom, float startAngle, float sweepAngle, boolean useCenter, Paint paint)
绘制指定的圆弧，该圆弧将被缩放以适应指定的椭圆

drawArc(RectF oval, float startAngle, float sweepAngle, boolean useCenter, Paint paint)
绘制指定的圆弧，该圆弧将被缩放以适应指定的椭圆
（userCenter说明:如果画笔是填充模式paint.setStyle(Paint.Style.FILL)，userCenter=true时将绘制一个扇形，false时将绘制一个弧形； 如果画笔是画线模式paint.setStyle(Paint.Style.STROKE)，userCenter=false绘制不封口弧线）

通过path绘制

drawPath(Path path, Paint paint)
使用指定的Paint对象绘制指定的路径。

绘制位图

drawBitmap(Bitmap bitmap, Matrix matrix, Paint paint)
使用指定的矩阵绘制位图

drawBitmap(Bitmap bitmap, Rect src, Rect dst, Paint paint)
绘制指定的位图，自动缩放/转换以填充目标矩形。

drawBitmap(Bitmap bitmap, float left, float top, Paint paint)
绘制指定的位图，它的顶部/左侧角(x,y)，使用指定的颜色，由当前矩阵变换。

绘制文本

drawText(CharSequence text, int start, int end, float x, float y, Paint paint)
使用指定Paint对象在(x,y)处绘制指定范围的文本

drawText(String text, float x, float y, Paint paint)
使用指定Paint对象在(x,y)处绘制指定文本

drawText(char[] text, int index, int count, float x, float y, Paint paint)
使用指定Paint对象在(x,y)处绘制指定文本

drawText(String text, int start, int end, float x, float y, Paint paint)
使用指定Paint对象在(x,y)处绘制指定文本

drawTextOnPath(String text, Path path, float hOffset, float vOffset, Paint paint)
在指定的路径上，使用指定的Paint对象绘制文本。

drawTextOnPath(char[] text, int index, int count, Path path, float hOffset, float vOffset, Paint paint)
在指定的路径上，使用指定的Paint对象绘制文本。
